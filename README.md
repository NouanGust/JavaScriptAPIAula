# TudoSobreGatos

Projeto desenvolvido durante as aulas do **RenovaTech**, um programa de capacitação em tecnologia e desenvolvimento de software.

## Sobre o Projeto

Aplicação web que consome duas APIs públicas para exibir fatos e fotos aleatórios sobre gatos. Ao clicar no botão "Descobrir", um fato e uma imagem são buscados em tempo real e apresentados na tela.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)

## APIs Utilizadas

- [Cat Fact Ninja](https://catfact.ninja/) — fornece fatos aleatórios sobre gatos
- [The Cat API](https://thecatapi.com/) — fornece imagens aleatórias de gatos com metadados de dimensões

## Temas Abordados

### JavaScript e Consumo de APIs
- Requisições HTTP com `fetch`
- Uso de `async/await` para programação assíncrona
- Tratamento de respostas JSON
- Lógica de repetição com `do...while` para filtrar resultados por tamanho

### Manipulação do DOM
- Seleção de elementos com `getElementById` e `querySelector`
- Atualização dinâmica de conteúdo com `innerText` e atributos como `src`
- Modificação de estilos via JavaScript (`element.style`)
- Eventos de clique com `addEventListener`

### HTML
- Estrutura semântica de página
- Vinculação de folha de estilos e script externo
- Uso de favicon

### CSS
- Flexbox para layout centralizado
- CSS aninhado (nested CSS)
- Unidades responsivas com `clamp()`
- Efeitos visuais com `box-shadow` e `text-shadow`
- Interatividade com pseudo-classe `:hover`

## Como Executar

Abra o arquivo `index.html` diretamente no navegador. Nenhuma instalação é necessária.
